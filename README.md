# Wedding Invitation Backend

Node.js backend server for the heytext website.

## Features

- 📡 RESTful API endpoints
- 🔒 Secure data handling
- 📝 RSVP management
- 💭 Wishes wall functionality
- 📸 Photo gallery management
- 🔄 CORS configuration
- 📊 Basic analytics

## Tech Stack

- Node.js
- Express.js
- MongoDB
- PM2 (Process Manager)
- Nginx (Reverse Proxy)
